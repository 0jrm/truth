def parse_note(text: str, path: Path | None = None) -> ParsedNote:
    """split + validate (no auto_type — missing type raises ValueError)."""

Rules:
- `yaml.safe_load` only
- After load, `isinstance(meta, dict)` or raise `ValueError("frontmatter must be a YAML mapping")`
- Missing/empty `type` + `auto_type=None` → `ValueError("missing required frontmatter field: type")`
- Missing/empty `type` + `auto_type="Note"` → inject type
