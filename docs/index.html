Architecture

notes/*.md ──► agent (memory_write) ──► log.md ▲ │ │ ▼ └── watcher ──► indexer ──► memory.db (SQLite) │ memory_search ◄───┘

Agent tools

  • memory_search — hybrid vector + BM25 (RRF), optional type/tags filters
  • memory_write — OKF markdown with YAML frontmatter + changelog
  • memory_delete — remove notes; watcher cleans the index

Stack

  • Python 3.11+ · sqlite-vec · sentence-transformers (local ONNX)
  • watchdog file watcher · MCP stdio server for Cursor / Claude Code
